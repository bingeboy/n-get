#n-get

Goal to create wget in nodejs.

### How To Install
```
$npm install n-get -g

```
### How To Use
```
$nget [protocal]filePath [WritePath]

```
* If no protocal is used in the file path http will be used by default
* If no writePath is provided http will be used by default
